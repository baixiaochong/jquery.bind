# jquery.bind 通过绑定一些自定义属性，自动绑定一个js事件，省去写js的烦恼。


### 1.返回顶部
1.添加自定义属性bind-gotop
2.添加css要样式

```html
<div bind-gotop class="gotop">GOTOP</div>
```
```css
.gotop {
    width: 80px;
    height: 80px;
    background: #222222;
    position: fixed;
    right: 30px;
    bottom: 30px;
    color: #ffffff;
    text-align: center;
    line-height: 80px;
    cursor: pointer;
}
```
